Piano!

ENG

Tijdens dit projectje heb ik vooral geexpirimenteert met geluid. Omdat geluid een belangrijke rol speelt bij games, wou ik meer weten over hoe het allemaal werkt achter de schermen. Bij dit project heb ik ook geexpirimenteert met het opnemen van je noten (als je deze optie aanklikt), zodat je later hierna kon luisteren. Desondanks ik niks met achterground muziek gedaan, heb ik toch geleerd hoe ik geluid kan toepassen voor objecten op meerdere manieren.
