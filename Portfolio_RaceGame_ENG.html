RaceGame project

nl

Ik heb bij dit project een kleine race game gemaakt, waarbij je alleen of met een andere op een map racen voor de beste tijd Bij dit project heb ik de beweging gemaakt, de splitscreen multiplayer en de speed boost die je op de maps kan tegen komen. De bedoeling van dit project was om een kleine game te maken in een korte periode.

Met dit project hebben ik gewerkt via de agile methode en heb ik meer ervaring gekregen met sprint planningen. Ook heb ik tijdens dit project mijn eerste versie van een co-op element toegevoegd aan de game
